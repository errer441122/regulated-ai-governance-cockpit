## Positioning

The UNDP-facing value is not model complexity. The value is a transparent programme-support case: public indicators, stakeholder review, capacity-building action, monitoring and evaluation thinking, and a boundary that prevents the tool from becoming automated aid allocation, eligibility, procurement, targeting, or policy decisioning.
