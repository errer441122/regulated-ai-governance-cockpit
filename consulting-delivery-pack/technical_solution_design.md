# Technical Solution Design

## Components

- Data validation: schema, numeric ranges, duplicate checks, and anomaly notes.
- Feature engineering: transparent tabular features and SQL mart contracts.
- Model baseline: Logistic Regression and RandomForest with calibration and drift artifacts.
- API simulation: FastAPI-compatible local scoring contract.
- Evidence reporting: reproducibility manifest, generated report, model card, data card, and limitations.

## Control Model

All model outputs remain advisory. Human review is required before operational action.
