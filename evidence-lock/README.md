# Evidence Lock v1.0

This folder is the permanent reviewer evidence bundle for the regulated AI governance cockpit.

## Reproduce

```bash
make setup
make evidence
```

The command regenerates tests, ML metrics, calibration/drift summaries, public-data lab artifacts, API smoke evidence, and the consolidated report at `results/portfolio_evidence_report.md`.

## Boundary

This is portfolio evidence only. It is not a production system, legal compliance tool, credit model, aid-allocation tool, or real deployment for CRIF, PwC, UNDP, BI-REX, CINECA, or IT4LIA.
