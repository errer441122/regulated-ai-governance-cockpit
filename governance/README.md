# Governance Audit Pack

This folder translates the portfolio evidence into a structured audit pack.

Sources used:

- NIST AI RMF 1.0: `https://www.nist.gov/itl/ai-risk-management-framework`
- NIST AI RMF Playbook: `https://www.nist.gov/itl/ai-risk-management-framework/nist-ai-rmf-playbook`
- NIST AI RMF 1.0 publication: `https://www.nist.gov/publications/artificial-intelligence-risk-management-framework-ai-rmf-10`
- EU AI Act official text: `https://eur-lex.europa.eu/eli/reg/2024/1689/oj?locale=en`
- EU AI Act overview and implementation guidance: `https://digital-strategy.ec.europa.eu/en/faqs/navigating-ai-act`
- ISO/IEC 42001 public overview: `https://www.iso.org/standard/42001`

Main files:

- `nist_ai_rmf_mapping.csv`
- `eu_ai_act_gap_analysis.md`
- `iso42001_public_gap_analysis.md`
- `validation_checklist.md`
- `model_change_log.md`

Boundary: this is a portfolio gap analysis and evidence map, not legal advice, conformity evidence, certification evidence, or a proprietary ISO control checklist.
