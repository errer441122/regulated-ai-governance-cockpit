# NIST AI RMF Public Source Note

Source URL: https://www.nist.gov/publications/artificial-intelligence-risk-management-framework-ai-rmf-10

The NIST AI Risk Management Framework is a voluntary public framework for identifying and managing AI risks across the lifecycle. The local RAG note uses it as a governance reference for the functions Govern, Map, Measure, and Manage.

Trustworthy AI governance requires documented roles, risk tolerances, impact context, measurement procedures, monitoring cadence, and accountability for residual risk. A reviewer should expect evidence that the project maps system context before model evaluation and that it records what can and cannot be automated.

For this portfolio, the NIST-aligned control is evidence discipline: every model or RAG answer should have a source, a stated boundary, a human-review path, and a monitoring signal. This is not a NIST certification or an official NIST implementation.
