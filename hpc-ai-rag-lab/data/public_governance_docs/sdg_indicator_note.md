# SDG Indicator Public Source Note

Source URL: https://unstats.un.org/sdgs/indicators/indicators-list/

The United Nations Statistics Division maintains the public global indicator framework for the Sustainable Development Goals. This local note uses the official SDG indicator list as a reminder that public-development analytics should separate global indicators, national context, and local programme evidence.

An SDG analytics workflow should record indicator provenance, update date, geographic coverage, missingness, disaggregation limits, and whether an indicator is suitable for the decision being discussed. A dashboard should not imply causal impact from a single indicator movement.

For this portfolio, the SDG control is source humility: RAG answers can summarize the local evidence pack, but they must not invent country diagnostics, programme outcomes, or official UNDP positions.
