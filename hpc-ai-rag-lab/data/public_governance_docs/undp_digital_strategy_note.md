# UNDP Digital Strategy Public Source Note

Source URL: https://www.undp.org/publications/digital-strategy-2022-2025

UNDP's public Digital Strategy 2022-2025 frames digital transformation around public value, country support, inclusion, capability building, and responsible use of data and technology. This local note paraphrases the public strategy for a portfolio RAG workload.

For a public-sector AI or data project, the review should ask whether data use is proportionate, whether local context is represented, whether capability transfer is planned, and whether the output supports human programme decisions rather than replacing them.

For this portfolio, the UNDP-style control is a policy and delivery boundary: AI answers must cite local evidence, avoid unsupported claims about country conditions, and describe what a real country-office validation would still need.
