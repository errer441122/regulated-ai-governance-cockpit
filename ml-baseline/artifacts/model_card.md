# Regulated Risk ML Baseline Model Card

## Intended Use

Advisory human-review triage for a simulated regulated decision-support workflow. The model helps reviewers inspect feature engineering, metrics, calibration, and model governance artifacts.

## Out-Of-Scope Use

- Automated credit, eligibility, procurement, legal, aid allocation, or customer decisions.
- Production compliance certification.
- Real CRIF, PwC, UNDP, CINECA, IT4LIA, BI-REX, employer, customer, or public-sector deployment.

This baseline is not a fairness-certified or legally reviewed model. It is a portfolio simulation. Any real regulated use would require legal review, bias testing, data protection assessment, monitoring, stakeholder governance, and human accountability.
