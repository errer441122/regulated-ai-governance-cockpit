# UNDP Public Data GIS Lab

Offline-first public-data and GIS-lite lab for UNDP Digital, AI, and Innovation-style review. It uses a documented public-data framing, synthetic coordinates for country-level examples, lightweight SDG scoring, responsible-data controls, a policy note, and generated PNG outputs.

Boundary: briefing and portfolio evidence only; not a real UNDP country-office deployment, aid-allocation tool, procurement workflow, or eligibility model.

## Run

```bash
python undp-public-data-gis-lab/sdg_indicator_analysis.py
```
