# Public Data Source

Reference source: World Bank World Development Indicators and public SDG-style national indicators.

This repository keeps a tiny offline sample for reproducibility. The lab demonstrates the workflow shape a reviewer should look for: public-data source note, indicator normalization, GIS-lite coordinates, SDG framing, policy note, responsible-data checklist, and human-review boundaries.

No UNDP internal data, beneficiary data, procurement data, or country-office operational data is used.
